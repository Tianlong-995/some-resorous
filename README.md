## 写在前面

- ✨ 平时由于工作以及个人爱好需要收集了不少的文件资料，随着这些资料越收集越多，如何保存也是一个大问题，目前没有发现一个可以一劳永逸地保存所有资料的办法，毕竟云端网盘的容量有限，实体硬盘大多价格昂贵而且有损坏从而损失所有资料的风险！
- ✨ 但是有一样东西是永远不会变的，那就是相互分享的互联网精神！
- ✨ 是的，我认为个人只要保留一个索引，然后只要互联网还在，那么知识就会永远都在，也欢迎大家也积极贡献各种优质的资源，我会花时间整理成册，发布在这里！
